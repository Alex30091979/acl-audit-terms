## 2. Описание сервиса

**ACL Audit — Drive Permissions** — это расширение для браузера Google Chrome, которое предоставляет инструменты для аудита и управления правами доступа к файлам и таблицам Google Drive.

### Основные функции:
- Сканирование прав доступа к файлам Google Drive
- Поиск внешних пользователей и публичных доступов
- Управление правами доступа (отзыв прав)
- Экспорт результатов в Google Sheets
- Снапшоты и сравнения (PRO)
- Автоматическое сканирование (ULTRA)
- Аналитика и аудит (ULTRA)

---

## 3. Требования к использованию

### 3.1. Системные требования
- Google Chrome (последняя версия)
- Аккаунт Google Workspace или личный аккаунт Google
- Таблица Google Sheets для сохранения результатов
- Разрешения на доступ к Google Drive API и Google Sheets API

### 3.2. Авторизация
Для работы Расширения требуется авторизация через Google OAuth 2.0. Вы предоставляете Расширению доступ к:
- Google Drive API (чтение метаданных файлов и прав доступа)
- Google Sheets API (создание и редактирование таблиц)

## 5. Использование данных

### 5.1. Обработка данных
- Все данные обрабатываются локально в вашем браузере
- Результаты сканирования сохраняются только в вашей таблице Google Sheets
- Настройки и ключи лицензий хранятся локально в браузере (chrome.storage)
- Расширение не передаёт ваши данные третьим лицам

### 5.2. Доступ к данным Google
Расширение запрашивает доступ только к:
- Метаданным файлов Google Drive (название, права доступа)
- Возможности создания и редактирования таблиц Google Sheets

Расширение **НЕ** запрашивает доступ к:
- Содержимому файлов
- Личным сообщениям
- Другим данным вашего аккаунта Google

### 5.3. Безопасность
- Используется OAuth 2.0 для безопасной авторизации
- Все данные экранируются для защиты от XSS-атак
- Токены доступа хранятся безопасно и автоматически очищаются при истечении
